Location & User Types

Here you can assign your user to a store (or multiple stores). This will allow them to access the features for a particular store, depending on the user type you assign them.
User type determines which pages a user can access:

  • Owner: access everything in the administration panel
  • Manager: access most of the administration for the store(s) they are assigned to
  • Employee: access to inventory, point of sale, and pull list for the store(s) they are assigned to
